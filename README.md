#  NurseIt Study – Νοσηλευτικές Ερωτήσεις & Απαντήσεις

Διαδραστική web εφαρμογή επανάληψης για τις εξετάσεις **Βοηθού Νοσηλευτικής – Γενικής Νοσηλείας**.  
Περιλαμβάνει **305 ερωτήσεις** με απαντήσεις, οργανωμένες σε δύο ομάδες.

**Live demo:** [nurseit-study.vercel.app](https://nurseit-study.vercel.app)

---

##  Χαρακτηριστικά

- **Δύο ομάδες ερωτήσεων** — Ομάδα Α (256 ερωτήσεις) & Ομάδα Β (49 ερωτήσεις)
- **Accordion UI** — οι απαντήσεις αποκαλύπτονται με κλικ στην ερώτηση
- **Responsive** — λειτουργεί σε κινητό και desktop
- **Keyboard accessible** — πλοήγηση με Enter/Space
- **Buy Me a Coffee** — υποστήριξη του δημιουργού

---

## Δομή αρχείων

```
├── index.html       # Σελίδα Ομάδας Α
├── group_b.html     # Σελίδα Ομάδας Β
├── script_a.js      # Δεδομένα & λογική Ομάδας Α (256 ερωτήσεις)
├── script_b.js      # Δεδομένα & λογική Ομάδας Β (49 ερωτήσεις)
└── style.css        # Κοινό stylesheet και των δύο σελίδων
```

---

##  Εκτέλεση τοπικά

Επειδή τα δεδομένα είναι embedded στα JS αρχεία, μπορείς απλά να ανοίξεις το `index.html` στον browser σου — δεν απαιτείται server.

```bash
git clone https://github.com/YOUR_USERNAME/nurseit-study.git
cd nurseit-study
# Άνοιξε το index.html στον browser σου
```
