#  NurseIt Study – Νοσηλευτικές Ερωτήσεις & Κουίζ

Διαδραστική web εφαρμογή επανάληψης για τις εξετάσεις **Βοηθού Νοσηλευτικής – Γενικής Νοσηλείας**.  
Περιλαμβάνει **305 ερωτήσεις** με απαντήσεις και **16 διαδραστικά κουίζ** (8 ανά ομάδα).

**Live demo:** [nurseit-study.vercel.app](https://nurseit-study.vercel.app)

---

##  Χαρακτηριστικά

### Ερωτήσεις & Απαντήσεις
- **Δύο ομάδες ερωτήσεων** — Ομάδα Α (256 ερωτήσεις) & Ομάδα Β (49 ερωτήσεις)
- **Accordion UI** — οι απαντήσεις αποκαλύπτονται με κλικ στην ερώτηση
- **Responsive** — λειτουργεί σε κινητό και desktop
- **Keyboard accessible** — πλοήγηση με Enter/Space

### Κουίζ & Αξιολόγηση
- **16 διαδραστικά κουίζ** — 8 ανά ομάδα
- **Επιλογή κουίζ** από dropdown menu
- **Εμφάνιση ερωτήσεων** μία προς μία με 4 επιλογές
- **Πλοήγηση** μεταξύ ερωτήσεων (προηγούμενη/επόμενη)
- **Αποτελέσματα** με ποσοστό επιτυχίας
- **Εξηγήσεις** για τις σωστές απαντήσεις
- **Progress bar** με ένδειξη προόδου

### Άλλα
- **Buy Me a Coffee** — υποστήριξη του δημιουργού
- Modern UI με gradient backgrounds
- Smooth animations & transitions

## Πώς να χρησιμοποιήσετε

### Ερωτήσεις & Απαντήσεις
1. Πατήστε **Ομάδα Α** ή **Ομάδα Β** στο μενού
2. Κάντε κλικ σε μια ερώτηση για να δείτε την απάντηση
3. Κάντε κλικ ξανά για να κρύψετε την απάντηση

### Κουίζ
1. Πατήστε το **hamburger menu (☰)**
2. Επιλέξτε **Κουίζ**
3. Επιλέξτε το κουίζ που θέλετε από τη λίστα
4. Πατήστε **Έναρξη**
5. Απαντήστε τις ερωτήσεις
6. Δείτε τα αποτελέσματα με αναλυτικές εξηγήσεις
